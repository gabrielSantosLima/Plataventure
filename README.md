# Plataventure
Plataventure é um jogo de plataforma feito em Javascript utilizando a biblioteca p5.js. 

## Instalação

Extraia o pacote ``.zip`` deste  [repositório](https://github.com/gabrielSantosLima/Plataventure.git) e execute o arquivo ``index.html`` no diretório raiz da aplicação.

## Mais informações

O desenvolvimento e planejamento do projeto será registrado em [nossa página](https://www.notion.so/Plataventure-2407a5931ac8450c83b0ede20c865ace) no Notion.
